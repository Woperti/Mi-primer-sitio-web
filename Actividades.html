Actividades

  • 1. Discutir sobre la actividad en clase
  • 2. Formar equipo y debatir sobre un tema de interes
  • 3. Dividir el trabajo y planificar usando Trello o una tabla en excel
  • 4. Empezar el desarrollo del sitio web, creando un mockup(dibujo de como se vera la informacion que el equipo acordo)
  • 5. Crear un primer sitio web de prueba, realizar un pequeno test de usabilidad. seguir el siguiente enlace Usability test
  • 6. Seguir iterando hasta que el sitio web sea funcional y el usuario pueda entenderlo e interactuar para lograr el objetivo que se planteo, el cual resolvera un problema.
